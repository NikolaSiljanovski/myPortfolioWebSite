# Nikola Siljanovski - Portfolio

Personal portfolio website showcasing my projects, skills, and experience as a Full-Stack Developer & QA Engineer.

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd code-canvas-pro

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Features

- Responsive design
- Dark/Light mode support
- Smooth animations
- Project showcase
- Skills & experience timeline
- Contact section
